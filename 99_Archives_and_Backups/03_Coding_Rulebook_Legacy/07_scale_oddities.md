<summary><h2>Special Cases: Scale Oddities (Dummy & Proxy Variables)</h2></summary>

When coding effect sizes for variables that are not traditional multi-item psychological scales, standard psychometric properties cannot be extracted.

> [!IMPORTANT]  
> **Dummy and Proxy Variable Rule**  
> If a variable is a binary indicator, a demographic dummy variable (e.g., Male/Female coded as 0/1), or an objective proxy measure (e.g., Firm Size, ROA, Yes/No questions), psychometric reliability properties do not apply.
> 
> **Action:** You must code `999` for both the **Number of Items** and the **Reliability (Cronbach's Alpha)** for that specific variable, as it is mathematically impossible or theoretically irrelevant for these types of single-item or objective indices to have an Alpha.
